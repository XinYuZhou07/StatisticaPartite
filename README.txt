1. TRACCIA DA CONSEGNARE AGLI STUDENTI
PROGETTO: Prevedere l’imprevedibile – Probabilità nello sport
Obiettivo
Realizzare una web app che stimi la probabilità di vittoria o sconfitta di una squadra sulla
base di dati reali.

Lavoro di gruppo
• Gruppi da 3 studenti
• Ogni gruppo sceglie uno sport (es. calcio, basket, tennis...)

FASE 1 – Raccolta dati
Scegliete un campionato (es. Serie A, NBA) e raccogliete dati di almeno 30 partite.
Dati minimi:
• Squadra
• Avversario
• Risultato (vittoria/sconfitta)
• Punteggio
Formato: CSV o Excel

FASE 2 – Analisi matematica
Calcolate:
• Frequenza relativa delle vittorie
• Probabilità statistica

FASE 3 – Modello predittivo
Create una formula che stimi la probabilità di vittoria.
Dovete:
• spiegare la formula

• giustificare le scelte

FASE 4 – Web App
Realizzate una semplice applicazione web che:
• permetta di inserire i dati
• calcoli la probabilità
• mostri il risultato
Tecnologie consigliate:
• HTML / CSS / JavaScript
oppure Python

FASE 5 – Analisi critica
Spiegate:
• limiti del modello
• perché non è sempre affidabile
• cosa manca per migliorarlo

FASE 6 – Presentazione
Durata: 10 minuti
Dovete mostrare:
• dati
• modello
• app funzionante

CONSEGNE
• Dataset
• Relazione (max 5 pagine)
• Codice
• Presentazione

VALUTAZIONE
• Matematica
• Funzionamento app
• Chiarezza
• Lavoro di gruppo
• Originalità
